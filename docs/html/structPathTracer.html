Detailed Description

A simple path-tracing renderer.

The algorithm implemented here allows the caller to tune number of rays thrown at each iteration, as well as the maximum depth. It implements Russian roulette, so in principle it will take a finite time to complete the calculation even if you set max_depth to math.inf.
